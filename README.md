# AI Ops Support Console

A local AI-powered IT/Ops support assistant built with:

- Python + Flask
- Local LLM (Ollama)
- System diagnostics
- Decision logic for hardware/software issues
- Web-based conversational interface

## Features

- Collects system diagnostics (CPU, Disk, Network)
- Uses AI to summarize issues and suggest fixes
- Skips diagnostics for hardware/external device cases
- Clean, structured output for ops teams
- Web-based chat interface

## Run Locally

```bash
python3 app.py
